## Behavior

Commit and push. No rebase.

1. Stage files individually by name.
2. Commit with title + bullet-point format (see `~/.claude/skills/git.md` for full commit rules).
3. Push to remote. Set upstream if needed: `git push --set-upstream origin <branch>`.
